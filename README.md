## Description

Touristic agencies catalogue

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ docker compose -f docker-compose.deps.yaml up
$ npm run dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
